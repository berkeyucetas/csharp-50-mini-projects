# C# 50 Mini Projects

Bu repository, **C# ve WinForms öğrenme sürecimde geliştirdiğim 50 mini projeyi** içermektedir.
Projeler başlangıç seviyesinden orta seviyeye kadar ilerleyen bir yapıda hazırlanmıştır.

Amaç:

* C# programlama pratiği yapmak
* Algoritma becerilerini geliştirmek
* WinForms masaüstü uygulama geliştirmeyi öğrenmek
* GitHub üzerinde düzenli proje paylaşımı yapmak
